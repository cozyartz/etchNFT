## Welcome to EtchNFT

EtchNFT is a cutting-edge platform that bridges the gap between digital and physical by transforming your NFTs into beautiful, laser-etched collectibles. Whether you're a collector, artist, or project founder, EtchNFT makes your on-chain identity tangible.

## Technology Stack

- **Frontend**: Astro + React + Tailwind CSS
- **Backend**: Cloudflare Workers + D1 Database
- **Authentication**: Lucia + GitHub OAuth
- **Payments**: Square + Coinbase Commerce
- **Blockchain**: Wagmi + Viem + RainbowKit
- **Deployment**: Cloudflare Pages + Workers
